# 창원버스 INFORMATION 수집

## 개요

GitHub Actions 으로 자동 창원버스 API DATA 수집.
JavaScript와 HTML을 사용하여 위의 결과를 사용자 친화적인 방식으로 출력.

### 1. 세팅

DATA 수집 설정 : <br>
GitHub Repo Setting -> 공공 API 'SERVICE_KEY' : Secrets 키 설정
GitHub Actions 을 통해 busdatasave.py 를 실행
레포지토리 '창원버스 API' 데이터(json) 푸시
